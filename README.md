## 🏛️ Project Overview

**Maison Liann Estates** is an advanced academic web development project designed to simulate a high-end, luxury real estate portal for residential and commercial leasing. 

This project demonstrates proficiency in modern frontend architecture, UI/UX design principles, and dynamic DOM manipulation, serving as a comprehensive showcase of web development capabilities.

## ✨ Technical Features & Implementations

* **🌗 Adaptive Theming Architecture:** Engineered a fluid transition between a classic *Light Mode* and a modern *Dark Mode* utilizing CSS variables and JavaScript event listeners.
* **📖 Dynamic Fractional Pagination:** Custom-built listing navigation logic (e.g., `01 / 10`) capable of asynchronously rendering and filtering an array of 120 generated property objects without page reloads.
* **🎬 Interactive Modals & Media:** Developed frictionless modal overlays for contact inquiries, detailed property specifications, and embedded cinematic walkthroughs.
* **⚡ Scroll-Triggered UI Rendering:** Implemented the `IntersectionObserver` API to execute smooth, cubic-bezier fade-in animations as the user navigates the DOM.
* **📱 Responsive Layout Engineering:** Utilized CSS Grid and Flexbox to ensure structural integrity across mobile, tablet, and desktop viewports.

## 💻 Tech Stack

This project was developed strictly using native web technologies to ensure maximum performance and zero dependency bloat.

| Technology | Application |
| :--- | :--- |

| **CSS3** | Custom variables, Grid/Flexbox layouts, glassmorphism, and keyframe animations. |
